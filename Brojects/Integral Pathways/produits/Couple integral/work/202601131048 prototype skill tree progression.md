### Mécaniques Détaillées du Système XP dans "Tissage Intégral du Couple"

Voici un prototype complet et détaillé des mécaniques XP, conçu pour être addictif, équilibré et évolutif. Il s'aligne sur notre arbre de progression : orange (récompenses externes pour motivation immédiate), vert (mix reward/réflexion pour guérison), jaune (cozy intrinsèque pour signification profonde). XP est la "monnaie" principale – gagné par actions, dépensé pour unlocks – avec feedback visuel (barres, animations) pour immersion.

Le système est non-linéaire : gagnez XP n'importe où dans l'arbre, mais quizzes recommandent priorités. Mode couple sync scores (moyenne XP pour unlocks communs). App MVP : Bubble pour tracking, notifications push pour streaks.

#### 1. **Gains d'XP : Sources et Valeurs**
XP gagné par actions mesurables, avec multiplicateurs pour engagement. Total max/semaine : ~500 (pour progression saine, ~1h/jour).

| Source d'XP | Description | XP Gagné | Multiplicateurs | Exemple |
|-------------|-------------|----------|-----------------|---------|
| **Pratique Complétée** | Finir un exercice (journal, dialogue, méditation). | 20-50 XP (basique : 20, avancé : 50) | x1.5 couple (sync partenaire) ; x2 streak quotidien | Jour 1 : Écoute active = 20 XP. |
| **Quizz/Réflexion** | Compléter quizz hebdo ou journal fin de branche. | 30 XP | x1.2 si insight partagé (couple/essaim) | Quizz déphasage = 30 XP + recommandation racine. |
| **Streaks** | Jours consécutifs d'activité. | 10 XP/jour + bonus (x2 jour 7) | Reset après 3 jours manqués | Streak 5 jours = 50 XP bonus. |
| **Feedback Partagé** | Partager insight en mode couple/essaim (note partenaire). | 15 XP | x2 si positif (rating >4/5) | "J'ai aimé ton écoute" = 15 XP chacun. |
| **Milestones** | Fin niveau branche (ex. : niveau 3 communication). | 100 XP | x1.5 si tous exercices faits | Badge "Résolveur Rapide" + 100 XP. |
| **Défis Bonus** | Défis spéciaux (ex. : appliquer pratique IRL). | 40 XP | x2 validé par partenaire | Photo preuve rituel soin = 40 XP. |

**Total Semaine Type** : 3 pratiques (90 XP) + quizz (30) + streak (50) = 170 XP base ; + couple/essaim = 300+.

#### 2. **Dépense d'XP : Unlocks et Personnalisation**
XP dépensé pour customiser l'arbre, favorisant choix et progression. Pas de "perte" – tout est utile.

| Coût XP | Unlock | Description | Exemple d'Utilisation |
|---------|--------|-------------|-----------------------|
| 50 XP | Extension Racine | Ajoutez pratique bonus à une racine (ex. : + exercice AT pour communication). | Racine 1 : + "CNV Avancé" pour niveau 2. |
| 100 XP | Sous-Classe | Débloquez chemin personnalisé (ex. : "Chemin Ombre" pour déphasage émotionnel). | Quiz détecte ombre → unlock gratuit teaser, full pour 100 XP. |
| 150 XP | Mode Cozy | Passez de badges shiny à symboliques (jaune : réflexion sans XP visible). | "Essaim Journal" : Entries libres, insights IA. |
| 200 XP | Essaim Externe | Invitez ami/partenaire pour co-tracking (sync arbres). | Partagez milestone, +20% XP mutuel. |
| 300 XP | Booster Global | x1.5 XP temporaire (1 semaine) pour toute l'arbre. | Pour accélérer orange → vert. |

**Économie Équilibrée** : Gagnez 200-400 XP/semaine ; dépensez 100-200 pour progression. Épargne pour boosts.

#### 3. **Progression et Feedback Loops**
- **Niveaux Arbre** : 3 par branche (1 : Orange quick win, 2 : Vert guérison, 3 : Jaune intégration). Fin racine = 300 XP → tronc unlock.
- **Feedback Visuel** : Barre XP arbre (ex. : 70% racine 1 = "Proche niveau 2 !"). Notifications : "Streak 3 jours – +10 XP !". Semaine summary : "XP gagnés : 250 ; Outcome : +30% écoute perçue".
- **Choix du Joueur** : 
  - **Liberté** : Choisissez branche (quizz suggère 2/3).
  - **Personnalisation** : Settings : "Mode Competitif" (leaderboards couple), "Mode Cozy" (journal privé sans XP).
  - **Adaptation** : Quizz hebdo recalibre (ex. : "Déphasage sexuel détecté – priorisez stimulation").
- **Fin de Progression** : Arbre complet (~10k XP) → "Mode Essaim Éveillé" (cozy perpétuel : défis monde, communauté Substack).

#### 4. **Implémentation et Test (MVP)**
- **App Flow** : Login → Quizz → Arbre interactif (drag/drop branches) → Pratique du jour → Log XP → Summary.
- **MVP Test** : Google Forms pour quizz prototype, Notion pour arbre statique. Beta : 10 couples Substack pour feedback.
- **Monétisation** : Core gratuit (racines orange), extensions payantes (5€/sous-classe, 29€ full arbre).
