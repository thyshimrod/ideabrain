#### Vision Générale : Une Approche Incrémentale et Multi-Dimensions pour le Couple Intégral
La vision que nous avons co-construite est celle d'un couple intégral comme un partenariat évolutif, où la relation n'est pas un "fix" statique mais un véhicule pour la croissance mutuelle et collective. Inspiré de Wilber (AQAL pour analyser les quadrants : intérieur/extérieur, individuel/collectif), Ucik (IRP pour pratiques relationnelles), et nos ajouts (Ennéagramme pour drivers/peurs, Spirale pour niveaux de valeurs, EDT pour stades d'ego, AT pour jeux toxiques), l'approche est holistique mais pragmatique. On part du principe que les couples d'aujourd'hui posent des questions concrètes (ex. : "Comment arrêter les disputes ?" ou "Pourquoi notre intimité s'essouffle ?") plutôt que abstraites ("Comment transcender l'ego ?").

- **Dimensions Clés** : Ombre (psychologique, projections), Corps (physique, somatisation), Sexe (énergétique, polarités), Mental (cognitif, narratifs), Relationnel (interpersonnel, valeurs), Systémique (extérieur, essaim). Ces dimensions s'interrogent pour résoudre problématiques, avec un focus sur le soin mutuel (prendre soin pour éviter burn-out).
- **Progression Incrémentale** : Non-linéaire, adaptée au rythme : Orange (quick wins pragmatiques contre toxicité, pour public efficacité-oriented), Vert (guérison empathique pour harmonie), Jaune (intégration holistique pour co-création avec le monde). Le double tranchant de la croissance individuelle est central : elle booste la compréhension des drivers, mais sans inclusion de l'autre, déphasage – d'où des pratiques mixtes (solo pour self-awareness, couple pour co-évolution).
- **But Ultime** : Transcender la "pensée plate" (simplifications impuissantes) pour un "essaim éveillé" – un réseau vivant où le couple influence le monde, antifragile aux crises (Taleb-inspired).

Cette vision est lisible : commencez par besoins immédiats (tableaux problématiques), évoluez via check-in (quizz pour ajuster), pour une transformation signifiant sans overwhelm.

#### Modus Operandi : Comment Opérer l'Offre
- **Personnalisation et Rythme** : Tout commence par quizz/check-in (comme discuté) pour doser savoir – ex. : orange reçoit quick wins AT, vert ombre Ennéagramme, jaune essaim AQAL. Rythme : Modules hebdo/mensuels avec pauses suggérées (ex. : "Si overwhelm, revenez à basics").
- **Individuel vs Couple** : Pratiques solo pour self (évite déphasage initial), puis couple pour partage (app sync scores). Essaim externe pour inclusion monde.
- **Implémentation** : Substack ancrage (articles teasers), Gumroad pour ventes, Bubble pour app gamifiée (quizz/XP). Promotion : Posts X/Reddit r/relationships pour problématiques pragmatiques.
- **Évolution** : Check-in recalibrent (ex. : quiz 14 jours : "Shifts détectés – nouveau module adapté"), évitant déversement.

#### Risques
- **Overwhelm Théorique** : Trop d'outils (AQAL/EDT) pour orange – risque dropout. Mitigation : Quizz dose, start basics.
- **Déphasage** : Croissance solo mène à frustration – risque rupture. Mitigation : 60% pratiques couple, alerts "Invite partenaire ?".
- **Niche Trop Petite** : Public jaune limité (5-10% dev perso) – risque ventes basses. Mitigation : Entrée orange pour élargir, puis upsell.
- **Manque de Réalisme** : Pratiques trop abstraites – risque "pas pour moi". Mitigation : Exemples concrets, testimonials.
- **Financier** : Coûts dev app (200-500€ MVP) sans ROI immédiat. Mitigation : Start avec PDF low-cost.
